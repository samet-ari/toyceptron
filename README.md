# Toyceptron

Toyceptron est une implémentation simple d’un perceptron multi-couches (MLP) en Python, réalisée sans aucune bibliothèque externe.

L’objectif du projet est de comprendre le fonctionnement interne d’un réseau de neurones en construisant :

- une classe `Neuron`
- une classe `Layer`
- une classe `Network`

Le projet implémente uniquement la propagation avant (forward pass), sans apprentissage ni rétropropagation.

Ce travail permet de mieux comprendre :
- les combinaisons linéaires (poids + biais)
- les fonctions d’activation (Sigmoid, ReLU, Threshold, Identity)
- la structure d’un réseau de neurones
- les principes de la programmation orientée objet (OOP)
